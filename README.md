# Sprite Generator

Generate sprites from multiple images! CSS export available.

![Tool UI](https://i.imgur.com/wnwh5dC.png)

This is a very simple tool, based in HTML and JS.

Included features:

-   Image download;
-   CSS export;
-   Padding;
-   Number columns.

Feel free to use in any context!
